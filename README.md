# SSR E-Commerce Product Management Dashboard

This project is a Server-Side Rendered (SSR) E-Commerce Product Management Dashboard built using Next.js.  
It allows admins to manage products, track inventory, monitor stock levels, and view analytics through a clean and responsive dashboard.
